jQuery - Dimensions

With jQuery, it is easy to work with the dimensions of elements and browser window

jQuery Dimension Methods

jQuery has several important methods for working with dimensions:

  • width()
  • height()
  • innerWidth()
  • innerHeight()
  • outerWidth()
  • outerHeight()

jQuery Dimensions

A imagem acima, ilustra que se definido o innerHeight ou innerWidth, a ocupação do espaço da área será coberta do inicio do conteúdo(elemento) até a borda.

Definido o outerHeight ou outerWidth, a ocupação do espaço da área será coberta do inicio do conteúdo(elemento) até é o padding, cobrindo também a borda do elemento.

jQuery width() and height() Methods

The width() method sets or returns the width of an element (excludes padding, border and margin).

The height() method sets or returns the height of an element (excludes padding, border and margin).
